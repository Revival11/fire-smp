revilang

Revilang adalah bahasa pemrograman khusus yang dirancang untuk pengembangan web, dengan fokus pada kemudahan dan fitur bawaan yang komprehensif. Bahasa ini menawarkan pendekatan unik dalam sintaks dan struktur pemrograman.

I/O (Input/Output):

Revilang menyederhanakan operasi input dan output dengan sintaks yang intuitif. Untuk menerima input atau menampilkan output di terminal maupun di console, cukup gunakan perintah sederhana berikut:

input::"masukan nama anda: ";
output::"hello world";

Perintah input:: digunakan untuk menerima masukan dari pengguna, sementara output:: digunakan untuk menampilkan pesan atau variabel.

Deklarasi Variabel:

Revilang memperkenalkan dua jenis deklarasi variabel yang berbeda:

const.tipe key = value;
flex.tipe key;
flex.tipe key = value;

- const: Digunakan untuk variabel yang nilainya tidak akan berubah (konstan).
- flex: Digunakan untuk variabel yang nilainya dapat diubah (fleksibel).
Kedua jenis ini mendukung berbagai tipe data seperti int(integer), float, str(string), arr(array/list), bool(boolean), dan obj(object).

Assigment Variabel:

Berbeda dengan bahasa pemrograman lain, Revilang menggunakan kata kunci 'set' untuk mengassign atau mengubah nilai variabel:

set key = assigmentvalue;
operator: =, +=, -=, /=, *=

operator untuk menambah atau mengurang 1: 
++ dan --

Operator ++ dan -- memungkinkan increment dan decrement dengan cara yang sama seperti di bahasa C/JavaScript.

function event conditional:
func @name #mouse::mouse_right #a::ctrl+a {
  if mouse {
    output::"right click!!";
  }
}

function event condisional menyediakan kondisi yang berasal dari input keyboard dan mouse. dengan kombinadi yang bisa diatur denhan mudah dan otomatis.

- Fungsi umum dengan parameter
- Fungsi template dengang dukungan argv dan regex
- Fungsi matematis dengan operasi acak
- Fungsi event conditional yang dapat merespon event mouse atau keyboard

Bagian frontend Revilang menawarkan API yang kuat untuk manipulasi DOM dengan sintaks yang intuitif. Fitur utama meliputi:

  • Seleksi elemen berdasarkan ID, class, tag, atau path
  • Pembuatan elemen baru
  • Manipulasi struktur DOM
  • Pengaturan gaya dan template

Backend:

  run@send; //memanggil fungsi send di frontend. itu bisa juga di taruh di frontend, untuk memanggil fungsi sebaliknya.
}

app.publicData {
  //berisi beberapa variable data yang akan dikirim ke sisi frontend, dan bisa sebliknya
}

app.idmyrouteid {
  run@send; //route berdasarkan id
}

app.run {debug=true, host=localhost, port=5000};
app.render::'index.html';

Backend Revilang menyediakan cara sederhana untuk:

  • Mendefinisikan rute
  • mendefinisikan rute dengan id
  • Menjalankan aplikasi dengan konfigurasi cepat
  • Merender halaman HTML

Catatan: Untuk menggunakan Revilang, pastikan mengunduh bahasa dari website resmi.
